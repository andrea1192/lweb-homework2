Questo minisito contiene parte dei miei appunti per il primo modulo del corso "Linguaggi per il Web" (A.A. 2023/2024), trascritti in un formato più discorsivo ed aumentati con informazioni raccolte dalla documentazione di Mozilla. I contenuti sono navigabili utilizzando il menu a sinistra.

Funzionamento del sito

Ciascuna pagina è generata dinamicamente da uno script PHP, che carica la lista dei contenuti disponibili e quello richiesto da un database.

Il database è inizializzato dallo script install.php. Dopo essersi collegato con le credenziali inserite in connection.php, questo crea le tabelle necessarie ed utilizza una lista di contenuti per popolarle. Per impostazione predefinita, questa lista è integrata nello script ed include contenuti prelevati dalla cartella static.
